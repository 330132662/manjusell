# UI 提示词参考

### 设计一套AI短剧培训课程视频销售系统的UI界面   核心设计目标
受众：想学习AI工具（Sora、Pika、剪映等）创作短剧的普通创作者或影视从业者。

任务：通过视觉语言，传达“低门槛创作、AI赋能、高收益变现”的核心价值观。

风格关键词：科技感、极简、教育感、高转化、蓝紫渐变、粒子特效、3D元素。
主页 / 课程落地页（Landing Page）
作用：吸引眼球，展示课程卖点，引导注册。

Prompt:
Architecture & Layout: A sleek, high-converting SaaS landing page design for an AI short drama training course. The layout features a full-width hero section with a prominent "Start Learning" call-to-action button, followed by a scrollable grid of course modules.

Visual Elements: Central focus on a large, glowing 3D holographic display of AI-generated short drama characters floating above a laptop. Background particles forming neural network connections. Use a deep tech color palette: Midnight Blue (#0B132B) as primary, Neon Cyan (#00F2FE) and Purple (#4F46E5) for gradients.

Content Blocks: The main navigation bar includes "Courses", "AI Tools", "Case Studies", and "Pricing". Below the hero, display 3 feature cards: "AI Scripting", "Digital Humans", "One-click Distribution". At the bottom, show a "Top Student Works" carousel.

Text Style: Bold sans-serif headlines like "Create Viral Short Dramas with AI". High-contrast white text on dark backgrounds. Badges showing "Updated for 2026" and "30,000+ Students".

Design Language: Glassmorphism effects, rounded corners, subtle grid lines, and glowing hover states.

###  提示词 1：资源库总览页（Grid Gallery View）
目标：展示资源的丰富度和分类，引导用户浏览和筛选。

Prompt:
A high-converting SaaS resource library page for an AI short drama training course, featuring a "Scripts / Assets / Templates" repository. Deep tech color palette: Midnight Blue (#0B132B) background with Neon Cyan (#00F2FE) and Purple (#4F46E5) gradients.

Layout: Top section has a large hero banner with the headline "AI Short Drama Resource Arsenal" and a search bar with filter chips (e.g., "Scripts", "Digital Human Avatars", "Background Music", "Video Templates"). Below, display a grid gallery of resource cards (6 items visible, 2 rows of 3).

Resource Card Design: Each card includes a visually rich thumbnail (showing a scene from an AI-generated short drama), a title like "Cyberpunk City Chase Scene", tags: "#AI Script", "#Pika Style", a download/usage count badge (e.g., "Used 1.2k times"), and a prominent "Preview & Use" button with a gradient hover effect.

Special Elements: A floating "Hot Resources" carousel on the side or bottom, showing trending items with a glowing border. Include a "AI Generator" floating action button that suggests generating new scripts based on user input.

Design Language: Glassmorphism effects, rounded corners, subtle grid lines, glowing hover states, and a clean, organized feel. High-contrast white text on dark backgrounds. --ar 16:9 --v 6.0

🎬 提示词 2：资源详情预览页（Resource Preview & Download）
目标：展示资源的详细内容和价值，引导用户加入课程或下载。

Prompt:
A detailed resource preview page for an AI short drama training platform, focusing on a specific template or script. Split-screen layout: left side (60% width) features an interactive preview of the resource – for a script, show a formatted screenplay with scene descriptions and dialogue; for a video template, show a video player with timeline preview.

Right Sidebar: Displays resource metadata: Title, Category (e.g., "AI Script - Sci-Fi"), Difficulty Level (Beginner/Intermediate/Advanced), File Size, Format, and a "Download" or "Add to My Library" button. Below, show "Related Resources" recommendations as a horizontal scrollable list.

Trust Elements: Display a "Created by AI Expert" badge, a rating (4.9/5), and a "Used in 500+ Short Dramas" counter. Include a "Preview with AI Avatar" feature, where an AI character reads the script aloud.

Visual Elements: Use a dark, cinematic background for the preview area, with soft glows and shadows. The sidebar has a clean, white/glass background for readability. Smooth micro-interactions on hover. --ar 16:9 --v 6.0

⚙️ 提示词 3：资源上传与管理页（Instructor/Admin View）
目标：为讲师或管理员设计的上传和管理界面，体现后台的简洁高效。

Prompt:
A clean, functional Admin Dashboard interface for uploading and managing AI short drama resources (scripts, templates, assets). The layout features a left sidebar with "Resource Management" highlighted. Main area shows a data table with columns: Resource Name, Type, Upload Date, Status (Published/Draft), and Actions (Edit/Delete).

Top Action Bar: A prominent "Upload New Resource" button with a cloud icon. Filter dropdowns for "Category" and "Status". A search bar for quick lookup.

Upload Modal (Overlay): Show a floating modal window for uploading a new resource. It includes fields for: Resource Title, Description, Category (dropdown), Tags, File Upload (drag-and-drop area), Thumbnail Upload, and a toggle for "Free Preview" or "Paid Resource".

Visual Elements: White/light-grey background for clarity, with blue-purple brand accents on buttons and active states. Data-dense layout with clear typography and subtle borders. Icons from a professional icon set. --ar 16:9 --v 6.0
